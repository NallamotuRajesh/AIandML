Decision-Making Flows

Implement chains-of-thought and branching logic.

Chains-of-Thought

Break down reasoning into steps, passing intermediate results. Branching allows the model to consider multiple options.

Example

def decide(x):
    if x > 0:
        return "positive"
    else:
        return "non-positive"

print(decide(5))

Exercises

  • Extend decide to return a multi-step explanation of its decision.
  • Implement branching with multiple conditions (e.g., sign and parity).
  • Create a Mermaid diagram showing a three-way decision.
