Simulation & Training

Run simulated environments for agent training.

Environment & Rewards

Define states, actions, and rewards clearly. Simulation loops continue until a terminal state.

Example

for episode in range(3):
    state = env.reset()
    done = False
    while not done:
        action = agent.act(state)
        state, reward, done, _ = env.step(action)

Diagram

flowchart LR

    Reset --> AgentAction

    Environment --> Feedback
    Feedback --> AgentAction

Exercises

  • Modify the loop to log cumulative reward each episode.
  • Create a simple environment with only two states.
  • Experiment adding randomness to the agent's actions.
