Virtual Environments

Isolate dependencies per project to avoid conflicts and ensure reproducibility.

Example Commands

# Create environment
python -m venv venv

# Activate (Linux/macOS)
source venv/bin/activate

# Install packages
pip install pandas matplotlib

# Freeze requirements
pip freeze > requirements.txt
