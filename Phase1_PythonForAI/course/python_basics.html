Python Basics

Learn the foundation of Python: variables, data types, control flow, functions and simple I/O.

Example

# Variables & types
x = 10
name = "Alice"

# Control flow
if x > 5:
    print(f"{name} has a big number")

# Function
def greet(person):
    return f"Hello, {person}!"

print(greet(name))
