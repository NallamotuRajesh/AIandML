NumPy

NumPy provides fast array operations and is the backbone of scientific computing in Python.

Example

import numpy as np

# Create arrays
arr = np.array([1, 2, 3, 4])
matrix = np.zeros((2,2))

# Vectorized operations
print(arr * 2 + 5)

# Statistical
print(np.mean(arr), np.std(arr))
