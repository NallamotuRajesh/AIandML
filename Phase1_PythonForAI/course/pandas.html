Pandas

Pandas builds on NumPy to provide DataFrame objects for tabular data manipulation.

Example

import pandas as pd

# Load CSV
df = pd.read_csv('data.csv')

# Inspect
print(df.head())
print(df.describe())

# Clean
 df = df.dropna()

# Transform
df['total'] = df['a'] + df['b']
