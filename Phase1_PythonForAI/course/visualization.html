Data Visualization

Use Matplotlib and Seaborn to create plots that communicate insights clearly.

Example

import matplotlib.pyplot as plt
import seaborn as sns

x = [1,2,3,4,5]
y = [2,3,5,7,11]

plt.plot(x, y, label='primes')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Line Plot')
plt.legend()
plt.show()

sns.histplot(data=[1,2,2,3,3,3,4], kde=True)
plt.show()
