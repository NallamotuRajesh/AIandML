Jupyter Notebooks

Interactive environment combining code, text, and visualizations in cells.

Example

# In a notebook cell
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv('data.csv')
df.head()

# Next cell
plt.plot(df['x'], df['y'])
plt.show()
