# AI & Machine Learning Mastery Roadmap 🚀

A comprehensive 6-phase learning pathway to become an AI/ML engineer and architect, progressing from Python fundamentals to production-grade agentic AI systems. **Duration: ~17-20 weeks**

---

## 📋 Roadmap Overview

| Phase | Title | Duration | Focus | Capstone |
|-------|-------|----------|-------|---------|
| 1 | Python for AI | 2-3 weeks | Data fundamentals, Python ecosystem | **Student Analytics Engine** |
| 2 | Core Machine Learning | 4 weeks | Supervised/Unsupervised learning | **Credit Risk Prediction** |
| 3 | Deep Learning | 4 weeks | Neural networks, CNNs, RNNs | **Medical Image Classification** |
| 4 | NLP & LLMs | 4 weeks | Transformers, RAG, Embeddings | **Enterprise RAG Chatbot** |
| 5 | MLOps & Production AI | 3 weeks | Deployment, monitoring, scaling | **AI Deployment Platform** |
| 6 | Agentic AI Systems | 4 weeks | Multi-agent orchestration | **Multi-Agent Job Portal** |

---

## 🎯 Learning Outcomes

By completing this roadmap, you will:

✅ Master Python and data manipulation fundamentals (70% of ML work)  
✅ Build production-ready ML models with proper evaluation metrics  
✅ Design and train deep learning systems at scale  
✅ Understand LLMs, embeddings, and RAG architectures  
✅ Deploy AI systems using Docker, Kubernetes, and cloud platforms  
✅ Architect multi-agent AI systems aligned with 2026 best practices  
✅ Progress from **AI Engineer → AI Architect** career level

## 🚀 Quick Start

```bash
# Navigate workspace
cd /workspaces/AIandML

# Create virtual environment
python -m venv venv
source venv/bin/activate

# Start Phase 1
cd Phase1_PythonForAI/Capstone_StudentAnalytics
pip install -r requirements.txt
```

### 🛠️ Regenerate Dashboard
The central navigation page (`index.html`) is automatically generated based on the phase folders and
`LEARNING_GUIDE.md` headers. To rebuild it after adding or editing phases, run one of the following:

```bash
# use python interpreter explicitly
python scripts/generate_dashboard.py

# or execute directly if you added the executable bit
./scripts/generate_dashboard.py
```

The script overwrites `index.html` with a fresh dashboard layout including phase descriptions and links.
