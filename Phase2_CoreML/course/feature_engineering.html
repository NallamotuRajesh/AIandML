Feature Engineering

Construct new features from raw data to improve model performance.

Example

# create ratio feature
df['income_to_loan'] = df['income'] / (df['loan_amount'] + 1)

# binning
df['age_group'] = pd.cut(df['age'], bins=[0,30,40,50,100])
