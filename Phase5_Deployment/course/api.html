API & FastAPI

Expose models through RESTful endpoints using FastAPI.

Example

from fastapi import FastAPI
from pydantic import BaseModel
import joblib

app = FastAPI()
model = joblib.load("model.pkl")

class Input(BaseModel):
    features: list[float]

@app.post("/predict")
def predict(data: Input):
    return {"prediction": model.predict([data.features])[0]}

Advanced Notes

You can declare async endpoints, use dependency injection for preprocessing, and leverage BaseModel to validate inputs. Add CORS middleware for web clients.
