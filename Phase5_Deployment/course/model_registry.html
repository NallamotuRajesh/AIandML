Model Registry & Versioning

Track models, metadata and history (e.g., MLflow).

Example with MLflow

import mlflow
from sklearn.ensemble import RandomForestClassifier

with mlflow.start_run():
    model = RandomForestClassifier()
    model.fit(X_train, y_train)
    mlflow.sklearn.log_model(model, "model")
    mlflow.log_param("n_estimators", 100)

Key Concepts

  • Register: save model binary and metadata.
  • Stage: mark versions as production, staging, etc.
  • Serve: deploy specific versions via API.

Diagram

flowchart TD
    T[Training] --> R[Register Model]
    R --> V[Version Tracking]
    V --> D[Deployment]
    V --> S[Stage (prod/test)]

Exercises
